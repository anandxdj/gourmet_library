# 🍴 Gourmet Kitchen | Premium Recipe Discovery

A sophisticated, high-performance web application for discovering and exploring over 280+ gourmet recipes. Built with a focus on immersive UI, seamless navigation, and culinary inspiration.

## ✨ Key Features

- **Immersive Grid Gallery**: A high-density, visually rich display of the entire recipe collection.
- **Server-Side Pagination**: Optimized for performance, fetching 24 recipes per page for a buttery-smooth experience.
- **"Surprise Me" Discovery**: One-click access to random culinary inspiration via a dedicated randomizer.
- **Deep Detail Modals**: Immersive, full-screen recipe views that "clear" the UI to focus on preparation steps and ingredients.
- **Advanced Search**: Real-time filtering to quickly find specific meals or cuisines.
- **Premium Aesthetics**: A dark-themed, high-contrast UI utilizing **Inter** for precision and **Playfair Display** for elegance.
- **Fully Responsive**: Seamlessly scales from desktop workstations to mobile kitchen companions.

## 🚀 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS (Utility-first, high-performance)
- **Typography**: Google Fonts (Inter & Playfair Display)
- **API**: [FreeAPI](https://api.freeapi.app/) - Public Meals Database

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or bun

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   # or
   bun install
   ```
3. Run the development server:
   ```bash
   npm run dev
   # or
   bun run dev
   ```

## 📖 UI Design Principles

- **Focus**: The Modal architecture ensures that when you choose a recipe, everything else disappears so you can focus on cooking.
- **Contrast**: High-visibility typography ensures instructions are readable even from across the kitchen.
- **Subtlety**: Refined hover effects and smooth transitions provide a premium "hand-crafted" feel.
